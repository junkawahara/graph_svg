# DIMACS形式仕様

通常の DIMACS 形式とは仕様が異なる。
通常の DIMACS 形式では、頂点番号は 1 から n であるが、
ここでは、1 から n 以外も認める。

"p <n> <m>" のように、p から始まる行に
数字が 2 つ、スペース区切りで後から続くと、
それぞれ頂点数（ノード数）と辺数（エッジ数）を表す。

c から始まる行はコメントであり、無視する。

"e <src> <dest>" のように、e から始まる行は
辺（エッジ）を表す。スペース区切りで、
あとに続く文字列は 2 つの頂点を表す。

e の行に現れた頂点の種類数が n 未満のとき、
頂点数が n になるように、孤立頂点が存在することに注意。

改行は CRLF でも LF でもよい。

例
```
p 4 3
c This is a comment line.
e abc def
e def ghij3
e ghij3 45
```
abc, def, ghij3, 45 の4つの頂点からなるパス（辺の本数は3本）
